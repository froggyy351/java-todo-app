# 🚀 java-todo-app

**モモンガくんが迎えてくれる、爆速・軽量なCLIタスク管理ツール**

ターミナルの文字は全部白くて見づらい……そんな悩みをANSIカラーによる視覚的な色分けで解決しました。自分と他人のタスク（ボール）を一瞬で管理・切り替えできる、実用性重視のツールです。作者がいつもsakuraエディタ上で仕事中にやっていたことを、CLIでかっこよく効率よくできるようにしたいという想いから作成しました。

## 🐹 特徴
- **視認性バツグン**: ANSIカラーを駆使し、エラーは「レッド」、プロンプトは「ブルー」など、鮮やかな色分けで直感的に状況を把握できます。
- **モモンガくんの癒やし**: 起動するたびに、かわいいモモンガくんのアスキーアートがあなたを出迎えます。
- **堅牢なオートセーブ**: 操作のたびに `tasks.txt` へ保存。さらに都度ファイル保存するように配慮しているため、Ctrl+C等で強制終了した際もデータ損失を防ぎます。
- **超軽量**: 外部データベースは不要。テキストファイル一つで全てのタスクを管理します。
- **Java 8 互換**: 最新環境はもちろん、会社の古いJava 8環境でもそのまま動作するように設計されています。

## 🛠 使い方

### 1. 起動
配布されている `todo_v1.1.jar` があるディレクトリで以下のコマンドを実行します。
`java -jar todo_v1.1.jar`

### 2. メインメニュー
1. **[Create]**: 新しいタスクを追加します。
2. **[Read]**: 現在のタスク一覧を表示し、詳細操作モード（Edit Mode）へ移行します。
3. **[Update]**: タスクを完了状態（済）にします。
4. **[Exit]**: 終了します。
5. **[Read]**: 過去の完了済みタスク履歴を表示します。

### 3. リスト表示中の詳細操作 (Edit Mode)
リストを表示している間は、以下のショートカットコマンドで操作が可能です。
- `m[ID]` : 指定したIDのタスクを自分（ME）に設定します。 (例: `m3`)
- `[ID]=[Name]` : 指定したIDのタスクを別の人にパスします。 (例: `1=鈴木さん`)
- `d[ID]` : その場でタスクを完了（Done）にします。 (例: `d5`)
- `Enterキー` : メインメニューに戻ります。

## 📁 ファイル構成
.
├── src/           # ソースコード (.java)
├── bin/           # コンパイル済みクラスファイル
├── tasks.txt      # タスクデータ（自動生成されます）
└── todo_v1.1.jar  # 実行用JARファイル（完成品）

---

## 🔧 開発者向け：ビルド方法
もしソースから自分でビルド・JAR作成を行いたい場合は、以下の手順で行ってください。

1. コンパイル (src内のファイルをbinへ)
`javac -d bin -encoding UTF-8 src/*.java`

2. JAR作成 (binの中で実行)
`cd bin`
`jar -cvfe ../todo_v1.1.jar Main .`
